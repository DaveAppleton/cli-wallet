# CLIWallet

## Overview
The CLI Wallet app currently only supports the [local testnet](https://github.com/spacemeshos/local-testnet/).

## Building
with `go`
```
go get && go build
```
with `docker` 

`make dockerbuild-go`

## Using
Please follow the instructions in our [local testnet guide](https://testnet.spacemesh.io/#/local)
